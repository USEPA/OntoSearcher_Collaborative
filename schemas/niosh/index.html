How to Read This Schema

The interactive visualization shows the relationships between classes and properties in your RDF data:

  • Boxes (green): Represent classes
  • Circles (blue): Represent properties
  • Solid arrows: Indicate class hierarchy relationships (subClassOf)
  • Dashed arrows from class to property: Indicate that the property has that class as its domain
  • Dashed arrows from property to class: Indicate that the property has that class as its range

You can:

  • Drag nodes to rearrange the visualization
  • Zoom in/out using mouse wheel
  • Click on a node to highlight its connections
  • Use the physics controls to adjust the layout
